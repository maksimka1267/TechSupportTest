## Завдання 1 — Їжачок (C#)

У папці `HedgehogApp` реалізовано:

* клас `HedgehogSolver` з методом
  `MinMeetings(int[] counts, int desiredColor)`, де:

  * `counts[0]` — кількість **червоних** їжачків
  * `counts[1]` — кількість **зелених**
  * `counts[2]` — кількість **синіх**
  * `desiredColor` ∈ {0, 1, 2} — бажаний колір

Метод повертає **мінімальну кількість зустрічей** або `-1`, якщо отримати всіх їжачків одного заданого кольору неможливо.

Консольна програма:

* виводить кілька тестових прикладів;
* пропонує ввести кількість їжачків трьох кольорів;
* дозволяє обрати цільовий колір;
* виводить відповідь.

## Завдання 2 — Sudoku (JavaScript)

У файлі `JsSudoku/validSolution.js` реалізовано функцію:

```js
validSolution(board)
```

* `board` — масив 9×9 цілих чисел від 1 до 9;
* функція повертає `true`, якщо:

  * кожен рядок містить числа 1–9 без повторів;
  * кожен стовпець містить числа 1–9 без повторів;
  * кожен підквадрат 3×3 містить числа 1–9 без повторів.

Додано **консольне меню (Node.js)** з трьома режимами:

1. Використати готовий коректний приклад з умови.
2. Згенерувати випадкову матрицю 9×9.
3. Ввести матрицю вручну (9 рядків по 9 чисел).

Після вибору режиму програма виводить матрицю та результат `validSolution(board)`.

### Приклад запиту та результату

Запуск:

```bash
cd JsSudoku
node validSolution.js
```
